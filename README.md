## Project Run & Start Instructions

### Environment
- node.js 17 or higher version
- git
### Project construction
- Download dependency  
``` npm install ```
- Running  
``` npm run dev ```
- Build  
``` npm run build ```
- Test  
``` npm test ```
- Refresh the database
``` npm run wipe ```
